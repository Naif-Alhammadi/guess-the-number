# Gussing the Right Number Game
Fun game where the user enters a number to try and guess the correct one and collect points.

## Game Rules
There are three message that will apper :
1- Invalid Number: If the number is under 1 or above 20.
2- Too High: If the user's number is higher than the sectret number.
3- Too Low: If the user's number is lower than the sectret number.
